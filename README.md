# Customer Comment Livestream TikTok Classification
 Thu thập comment từ livestream TikTok.
- Phân loại comment vào 8 nhãn:
  1. Spam / Không liên quan
  2. Khen ngợi / Tương tác tích cực
  3. Hỏi thông tin sản phẩm
  4. Khẩn cấp / Cần phản hồi nhanh
  5. Hỏi vận chuyển / Thanh toán
  6. Hỏi giá
  7. Phàn nàn / Tiêu cực
  8. Chốt đơn / Mua hàng
- Hiển thị kết quả phân loại theo thời gian thực.

# Cấu trúc dự án

├─ api/ # Các endpoint API
├─ data/ # Dữ liệu huấn luyện / thử nghiệm
├─ modul/ # Mô hình phân loại và các hàm xử lý
├─ notebook/ # Các notebook thử nghiệm, demo
├─ static/ # CSS, JS, hình ảnh
├─ templates/ # Giao diện HTML
├─ requirements.txt # Thư viện cần cài đặt
└─ README.md
